##Running
Go to root folder of project, open your terminal and then follow steps:  
###Development Server  
- Installing dependencies:
```{r, engine='bash', count_lines}
$ npm install
```

```{r, engine='bash', count_lines}
$ bower install
```
- List Gulp tasks:  
```{r, engine='bash', count_lines}
$ gulp
```

- Running server:  
```{r, engine='bash', count_lines}
$ gulp serve
```
###Distribution Server  
- Generate distribution:  
```{r, engine='bash', count_lines}
$ gulp build
```

- Running distribution server:  
```{r, engine='bash', count_lines}
$ gulp dist:test
```
